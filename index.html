Qual a utilizade da programação ?

Ultimamente tem se reparado uma expansão no mercado da tecnologia, em específico voltado para a área da programação, mas a final de contas o que é programação e como ela interfere no meu dia a dia.
Programação consiste em pegarmos um problema ou necessidade do mundo real e através de algoritmos (sequência de instruções lógicas) criarmos uma solução. Na qual essa solução será transcrita para uma linguagem de computação, que consiste em uma forma de instruções específicas capaz de se comunicar com o computador, na qual o mesmo será responsável por armazenar informações, tratar e tomaras decisões de acordo com as diversas circunstâncias.

Entendendo sobre Cliente X Servidor

Precisamos entender dois conceitos importantes quando se refere a programação para Web, como exemplo imaginamos que ao abrir a página do Google nos deparamos com uma interface visual, contendo o logo da empresa e uma barra de pesquisa, podemos entender que a parte visual é o lado do cliente a qual chamamos de front end, e ao realizamos determinada pesquisa aquela informação digitada será encaminhada para um determinado servidor e essa informação será devolvida para o cliente, o processo de buscar informações no servidor de acordo com determinado tipo de dado se refere ao processo do back end, em outras palavras podemos dizer que o back end é o que acontece debaixo dos panos.

HTLM e algumas Tags

HyperText Markup Language também conhecido como HTML, consiste em uma linguagem de marcação utilizado na Web, um assunto muito polêmico seria dizer que o HTML não se classifica como linguagem de programação, pois diferente das linguagens como Java, C# e outras o mesmo não possui capacidade de tomar decisões lógicas e executar alguns loops e outras coisas comuns em linguagens de programação. O HTML é constituído por aberturas e fechamentos de tags, as tags consistem em códigos que irão estruturar nosso conteúdo, o documento HTML possuí 3 tags essenciais para seu funcionamento, sendo a primeira tag o próprio HTML, informando que está sendo carregado um conteúdo em HTML, na sequência temos a head, que seria como se fosse a cabeça do ser humano, ele contém meta informações e possibilidades de alterar o título e referenciar alguns estilos e importação de fontes, na sequência temos a tag body, essa tag é responsável por carregar tudo aquilo que vai ser exibido na página.
Podemos classificar as tags em dois tipos, as tags semânticas que tem um sentido já definido e são interpretadas de forma mais fácil pelo navegador além de possibilitarem uma certa acessibilidade para determinados grupos, e as tags não semânticas como as famosas div, atualmente estamos no HTML 5, a qual possuí uma diversidade de tags, buscando trazer mais sentido para cada estrutura que utilizamos em nosso documento, além de trazer uma imensa acessibilidade, a estrutura do HTML 5 é carregada pelo DOCTYPE HTML.
